Metric RDF1

This test evaluates whether an ontology is available in at least one recognized RDF serialization format.

What does the test do?

It sends a request to the ontology URI and inspects the content returned to determine if it is served using one of the accepted RDF formats: - RDF/XML - Turtle (TTL) - N-Triples - JSON-LD

Why is this important?

RDF serializations are the backbone of data representation on the Semantic Web. Ensuring that an ontology is available in one of these formats guarantees that it can be parsed, reused, and integrated by external applications and services.

How does the test work?

The system checks the content negotiation or the actual content returned by the ontology URI. If at least one valid serialization format is detected, the test passes. Otherwise, it fails.

Valid outcome

The test is considered successful if the ontology is accessible and returned in any of the specified RDF serializations.
